👀 ¿Has usado una página web?

¡Claro que sí! Cuando ves videos, juegas en línea o buscas algo en Google... ¡estás usando páginas web!

🧱 ¿Cómo están hechas?

Las páginas web están hechas con código. Es como usar bloques de LEGO pero para construir cosas que ves en internet.

📄 ¿Qué hay en una página?

Pueden tener texto, imágenes, botones y hasta juegos. Todo eso se pone usando HTML, CSS y JavaScript (como si fueran ingredientes mágicos).
